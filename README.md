# The-Wheel

Reinventing the wheel using AI — a small task wheel app.

## What it does

- Lets you add tasks.
- Draws each task as a segment on a colorful wheel.
- Spins and lands on a random task.

## Run locally

Because this is a static app, you can serve it with any simple web server:

```bash
python3 -m http.server 8000
```

Then open <http://localhost:8000> in your browser.
